## 第二維度：個體化進階參數分析（微觀比較）

### 2.1 四大關鍵參數計算

#### **參數一：Weight（負磅）**
```
Weight = 馬匹實際負磅（磅）
```
- **意義**：直接反映馬匹在賽事中的負重
- **應用**：配合統計表進行初步篩選

#### **參數二：WeightD（負磅距離乘積）**
```
WeightD = Weight × Distance
```
- **意義**：量化負磅在整個賽程中的累積影響
- **應用**：比較不同路程下的總體負擔

#### **參數三：Weight Ratio（負磅體重比）**
```
Weight Ratio = (Weight ÷ Horse Weight) × 100%
```
- **意義**：標準化的相對負擔指標
- **應用**：消除體重差異，比較相對負擔
- **基準**：一般範圍 9-13%

#### **參數四：WeightRD（累積相對負擔）**
```
WeightRD = (Weight ÷ Horse Weight) × Distance
```
- **意義**：最綜合的負擔指標，結合相對負擔與路程
- **應用**：終極比較工具，數值越低越有利
- **基準**：
  - 1000米賽事：80-120
  - 1200米賽事：100-150
  - 1600米賽事：130-200

### 2.2 進階參數應用場景

#### **場景一：同負磅不同體重比較**
- 馬匹A：130磅，體重1200磅
- 馬匹B：130磅，體重1080磅

**計算：**
- A馬 Weight Ratio = 130÷1200×100% = 10.83%
- B馬 Weight Ratio = 130÷1080×100% = 12.04%

**結論：**A馬相對負擔更輕，更有優勢。

#### **場景二：長途賽累積負擔比較**
- 1600米賽事
- 馬匹A：133磅，體重1150磅
- 馬匹B：130磅，體重1150磅

**計算：**
- A馬 WeightRD = (133÷1150)×1600 = 185.0
- B馬 WeightRD = (130÷1150)×1600 = 180.9

**結論：**B馬累積負擔較低，末段或更有優勢。

## 第三維度：年齡與成熟度分析（生涯週期評估）

### 3.1 年齡階段分類

#### **潛力新星（2-3歲）**
- **特徵**：數據有限，充滿潛力
- **分析重點**：進步趨勢
- **觀察指標**：
  - 負距是否持續縮短
  - 走位是否愈發成熟
  - 名次是否穩步提升

#### **巔峰戰將（4-5歲）**
- **特徵**：生理心理黃金期
- **分析重點**：穩定性與實力兌現
- **判斷標準**：如同時通過第一、二維度篩選，為當然主角

#### **沙場老將（6-10歲）**
- **特徵**：經驗豐富，需警惕機能下滑
- **分析重點**：狀態判斷
- **關鍵問題**：「寶刀未老」還是「力不從心」？

### 3.2 狀態評估方法

#### **進步趨勢分析（適用於潛力新星）**
```python
進步指數 = (最近三場名次變化 + 負距變化 + 走位改善) / 3
```

**計算示例：**
- 名次：8→6→4（進步分數：+2）
- 負距：5馬位→3馬位→1馬位（進步分數：+2）
- 走位：後上→中游→前列（進步分數：+1）
- 進步指數：(2+2+1)/3 = 1.67（正數為進步）

#### **穩定性評估（適用於巔峰戰將）**
```python
穩定性分數 = 100 - (近五場名次標準差 × 10)
```

#### **狀態判斷（適用於沙場老將）**
```python
狀態指標 = 近期最佳表現 - 近期平均表現
```
- 指標 > 0：寶刀未老
- 指標 = 0：穩定發揮
- 指標 < -2：力不從心

## 綜合評分系統

### 4.1 評分公式

```python
綜合評分 = (統計上名率 × 0.4) + (負擔指標 × 0.3) + (評分指標 × 0.2) + (年齡加成 × 0.1)
```

**各項計算：**

#### **統計上名率分數**
```python
統計分數 = 統計上名率 ÷ 100
```

#### **負擔指標分數**
```python
負擔分數 = max(0, (150 - WeightRD) ÷ 150)
```

#### **評分指標分數**
```python
評分分數 = min(1.0, 馬匹評分 ÷ 60)
```

#### **年齡加成係數**
- 潛力新星：0.8（潛力高但風險大）
- 巔峰戰將：1.0（最佳狀態）
- 沙場老將：0.9（經驗豐富但可能下滑）

### 4.2 評級分級

- **A級（80-100分）**：絕對主角，強烈推薦
- **B級（60-79分）**：有力競爭者，值得關注
- **C級（40-59分）**：配腳人選，謹慎考慮
- **D級（<40分）**：競爭力不足，不建議

## 完整分析流程

### 步驟一：數據收集與整理
1. 收集賽事基本資料（路程、場地、班次）
2. 整理各馬基本數據（負磅、體重、年齡、評分）
3. 查閱近期賽績（至少近三場）

### 步驟二：第一維度篩選
1. 確定賽事分類（短途/中長途，正常地/變化地）
2. 計算各馬負磅區間
3. 查詢對應統計上名率
4. 初步排除明顯劣勢馬匹

### 步驟三：第二維度精算
1. 計算四大參數（Weight, WeightD, Weight Ratio, WeightRD）
2. 對比條件相近馬匹的參數差異
3. 識別隱藏優勢或劣勢

### 步驟四：第三維度評估
1. 分類各馬年齡階段
2. 分析近期狀態趨勢
3. 調整潛力與風險評估

### 步驟五：綜合評分與排序
1. 計算各馬綜合評分
2. 按分數排序得出推薦次序
3. 標註特別關注事項

### 步驟六：最終建議
1. 確定主選、次選、三選
2. 識別潛在黑馬
3. 標示風險馬匹

## 實際應用注意事項

### 數據品質控制
- 確保負磅、體重數據準確
- 核實近期賽績的場地條件
- 注意騎師更換對表現的影響

### 特殊情況處理
- 初出馬：以訓練表現和血統為輔助參考
- 復出馬：減重年齡加成係數
- 跨班作賽：調整評分基準

### 模型限制
- 無法預測意外事件（如馬匹受傷、騎師失誤）
- 未包含步速分析和戰術因素
- 賠率市場效率會影響實際投資回報

### 持續改進
- 定期更新統計數據庫
- 追蹤預測準確率並調整權重
- 納入更多影響因素（騎師、練馬師、檔位等）
